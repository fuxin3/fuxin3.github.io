my-blog-build-remark
2024-07-16 17:50:46 # easy #hexo #github

分享一下使用github page和hexo搭建blog的过程

平台:linux

需要用到的工具:一个GitHub账号

Gihub注册

注册github

创建 repo

在 github 新建一个名为 <github-username>.github.io 的 repo

安装nodejs

在 linux 平台先安装 nodejs：推荐用conda安装
conda install conda-forge::nodejs

安装hexo

再在conda环境中用 npm 命令安装 hexo
npm install -g hexo-cli

Blog初始化

初始化：
hexo init my-blog

生成web前端

生成web前端：
hexo generate

本地启动

本地启动（本地启动后可在本地查看网页，可用于检查修改）：
hexo server

部署到github

修改_config.yml下的deploy中的github账号信息，之后可以部署到github上：
hexo deploy

然后就可以通过 http:// <github-username>.github.io 访问你的网站啦

新建文章

新发表一篇普通文章：
hexo new post my-blog-build-remark
